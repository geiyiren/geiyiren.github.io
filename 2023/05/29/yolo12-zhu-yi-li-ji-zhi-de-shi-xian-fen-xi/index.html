源码

注意力机制的实现代码如下:

  • ultralytics/nn/modules/block.py
class AAttn(nn.Module):
    """
    Area-attention module for YOLO models, providing efficient attention mechanisms.

    This module implements an area-based attention mechanism that processes input features in a spatially-aware manner,
    making it particularly effective for object detection tasks.

    Attributes:
        area (int): Number of areas the feature map is divided.
        num_heads (int): Number of heads into which the attention mechanism is divided.
        head_dim (int): Dimension of each attention head.
        qkv (Conv): Convolution layer for computing query, key and value tensors.
        proj (Conv): Projection convolution layer.
        pe (Conv): Position encoding convolution layer.

    Methods:
        forward: Applies area-attention to input tensor.

    Examples:
        >>> attn = AAttn(dim=256, num_heads=8, area=4)
        >>> x = torch.randn(1, 256, 32, 32)
        >>> output = attn(x)
        >>> print(output.shape)
        torch.Size([1, 256, 32, 32])
    """

    def __init__(self, dim: int, num_heads: int, area: int = 1):
        """
        Initialize an Area-attention module for YOLO models.

        Args:
            dim (int): Number of hidden channels.
            num_heads (int): Number of heads into which the attention mechanism is divided.
            area (int): Number of areas the feature map is divided.
        """
        super().__init__()
        self.area = area

        self.num_heads = num_heads
        self.head_dim = head_dim = dim // num_heads
        all_head_dim = head_dim * self.num_heads

        self.qkv = Conv(dim, all_head_dim * 3, 1, act=False)
        self.proj = Conv(all_head_dim, dim, 1, act=False)
        self.pe = Conv(all_head_dim, dim, 7, 1, 3, g=dim, act=False)

    def forward(self, x: torch.Tensor) -> torch.Tensor:
        """
        Process the input tensor through the area-attention.

        Args:
            x (torch.Tensor): Input tensor.

        Returns:
            (torch.Tensor): Output tensor after area-attention.
        """
        B, C, H, W = x.shape
        N = H * W

        qkv = self.qkv(x).flatten(2).transpose(1, 2)
        if self.area > 1:
            qkv = qkv.reshape(B * self.area, N // self.area, C * 3)
            B, N, _ = qkv.shape
        q, k, v = (
            qkv.view(B, N, self.num_heads, self.head_dim * 3)
            .permute(0, 2, 3, 1)
            .split([self.head_dim, self.head_dim, self.head_dim], dim=2)
        )
        attn = (q.transpose(-2, -1) @ k) * (self.head_dim**-0.5)
        attn = attn.softmax(dim=-1)
        x = v @ attn.transpose(-2, -1)
        x = x.permute(0, 3, 1, 2)
        v = v.permute(0, 3, 1, 2)

        if self.area > 1:
            x = x.reshape(B // self.area, N * self.area, C)
            v = v.reshape(B // self.area, N * self.area, C)
            B, N, _ = x.shape

        x = x.reshape(B, H, W, C).permute(0, 3, 1, 2).contiguous()
        v = v.reshape(B, H, W, C).permute(0, 3, 1, 2).contiguous()

        x = x + self.pe(v)
        return self.proj(x)

qkv提取与拆分

B, C, H, W = x.shape
N = H * W 
qkv = self.qkv(x).flatten(2).transpose(1, 2)
if self.area > 1:
    qkv = qkv.reshape(B * self.area, N // self.area, C * 3)
    B, N, _ = qkv.shape
q, k, v = (
    qkv.view(B, N, self.num_heads, self.head_dim * 3)
    .permute(0, 2, 3, 1)

)

  • all_head_dim

    这个是保证输出的维度为分支头个数的整数倍的3倍

  • self.qkv()

    self.qkv()是一个1x1卷积，目的是将输入的张量转换为多头的qkv张量，所谓的多头注意力机制就是将输入张量进行卷积，进行维度变换。self.qkv()将一个输入张量”拆分“为多头的{q,k,v}”拼接向量“，然后再执行拆分。

    经过self.qkv()后维度由[B, C, H, W]转换为[B, 3*C, H, W]

  • flatten(2).transpose(1, 2)

    将张量的最后2个维度合并后将交换第1个通道和第2个通道。此时输出的维度为[B, H* W, 3*C]

  • qkv.reshape()

    将 原来的图像每张切成 area 份，所以总的 batch 就变成了 B × area

    每一份区域中，包含的 token 数是 $\frac{HW}{\text{area}}$，即空间被均匀划分。这里的token就是像素块，HW的图像被均匀的划分为area个像素块，那就是类似下面这样的:

    每个 token 的特征维度还是 3C（因为包含 q, k, v）,这里其实是张量的”深度“为3C

  • q, k, v提取

    • view

      qkv的维度为[B, N, num_heads * head_dim * 3]，这里将 qkv重新 reshape 成一个 4 维张量，每个位置上的 QKV 被合并在 head_dim * 3 上，形状变为: [B, N, num_heads, 3 * head_dim]

    • permute

      将维度重新排列，得到[B, num_heads, 3 * head_dim, N]新维度

    • split

      在第 2 维（即 3 * head_dim 这一维）上，按照每 head_dim 切 3 段，即：

      • 第一段：query $q$ → [B, num_heads, head_dim, N]
      • 第二段：key $k$ → [B, num_heads, head_dim, N]
      • 第三段：value $v$ → [B, num_heads, head_dim, N]

注意力得分计算

attn = (q.transpose(-2, -1) @ k) * (self.head_dim**-0.5)

维度变化：

  • q.transpose(-2, -1) → [B, num_heads, N, head_dim]

  • k → [B, num_heads, head_dim, N]

  • 相乘：矩阵乘法：
    $$
    [B, h, N, d] \times [B, h, d, N] → [B, h, N, N]
    $$

  • 最终得到：

    attn.shape = [B, num_heads, N, N]

每个位置 i 与所有 j 之间的相似度打分（即 attention 分数）

为什么乘以 $\text{head_dim}^{-0.5}$？

  • 为了缩放，使得 dot product 不会太大或太小，防止 softmax 梯度消失或爆炸。
  • 来源于原始 Transformer 论文中的 scaled dot-product attention。

softmax 获取注意力权重

attn = attn.softmax(dim=-1)
  • 对最后一维（即 token 维）做 softmax
  • 每一行是该 token 对所有其他 token 的权重和为 1

用注意力权重加权 value 向量

x = v @ attn.transpose(-2, -1)

维度说明：

  • v.shape = [B, num_heads, head_dim, N]
  • attn.transpose(-2, -1).shape = [B, num_heads, N, N]

乘法结果：
$$
[B, h, d, N] \times [B, h, N, N] → [B, h, d, N]
$$
输出的 x 就是每个 token 经过 weighted sum 的结果：

每个位置 i 的新表示，是用所有 j 的 value 加权加起来的结果。

还原 token 维度顺序

复制编辑
x = x.permute(0, 3, 1, 2)
  • 原始：[B, num_heads, head_dim, N]
  • 目标：[B, N, num_heads, head_dim]

将 x 排列回正常的 token 顺序，为后续拼接使用。

同步转换 value 的顺序（用于后面位置编码）

python


复制编辑
v = v.permute(0, 3, 1, 2)
  • 同样地把 v 从 [B, num_heads, head_dim, N] 转换为 [B, N, num_heads, head_dim]，以便后续 reshape 成图像格式。

最终 x 和 v 的形状变为：

python复制编辑x.shape = [B, N, num_heads, head_dim]
v.shape = [B, N, num_heads, head_dim]

接下来，它们会被 reshape 回 [B, C, H, W] 格式，然后用于加位置编码等操作。

反向 reshape 成原始 batch

x = x.reshape(B // self.area, N * self.area, C)
v = v.reshape(B // self.area, N * self.area, C)
B, N, _ = x.shape

这样操作是为了将 token 恢复回原始图像结构

恢复为图像格式

x = x.reshape(B, H, W, C).permute(0, 3, 1, 2).contiguous()
v = v.reshape(B, H, W, C).permute(0, 3, 1, 2).contiguous()

reshape(B, H, W, C) → 空间上恢复为 2D 格式

permute(0, 3, 1, 2)→ 转为常规图像格式[B, C, H, W]

contiguous() → 保证内存布局正确（防止后续错误）

添加位置编码

x = x + self.pe(v)

self.pe是一个分组卷积(每个输出通道用一个自己的卷积核处理输入通道)

用 v来生成相对位置编码，代表空间感知或局部增强，然后直接做逐位置加法

融合多头

self.proj(x)

多头注意力机制中每个 head 学习不同的子空间信息。self.proj(x) 是一个 1×1 卷积层，用于将多头注意力后的特征图从多头拼接的维度 (all_head_dim) 映射回模型原始的维度 dim，完成通道融合。
