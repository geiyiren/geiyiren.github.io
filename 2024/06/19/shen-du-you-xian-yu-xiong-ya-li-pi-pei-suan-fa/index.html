本文所有内容均以下面这幅二分图为例。

图的深度遍历

图的遍历，从某一点出发，访问所有顶点，且每个顶点只被访问一次。当然访问过程中必须要有连线才行。比如从0出发，只能访问4和6，不能访问5。

比如上面这幅图是一个动物园的地图。那么一个最简单的思路，就是随便选一个顶点，比如从0（出发点就类似于动物园大门）出发。进门后的路线设计有两种方案，一种就是0–4–1–5这样一直闷着头往前走看动物，到头了再回来。另一种方案就是先把0临近的4和6的动物们先看了，再往深处走看离得远的其他动物。

可以看出第一种路径越走越深入，当出现岔路的时候，优先往更远的地方走，因此被叫做深度优先遍历（DFS）。第二种是当出现岔路的时候，先把离得近的都访问了，走法比较发散，被叫做广度优先遍历（BFS）。

本文研究的是深度优限遍历。

深度优先遍历的流程就是，优先往深的地方走，没有路了，退回到最近的节点，往该节点其他岔路走。一种实现方式是递归。递归这个概念一开始比较难理解。我先具体的写一下上面这幅图的深度遍历流程。

  1. 从0出发。即访问0。
  2. 04有连线。4没被访问过，访问4。
  3. 40有连线。0被访问过。
  4. 41有连线，1没被访问过，访问1。
  5. 14有连线，4被访问过。
  6. 15有连线，5没被访问过，访问5。
  7. 51有连线，1被访问过。
  8. 退回1。（14、15在步骤5、6处理过了）
  9. 退回4。（40、41在步骤3、4处理过了）
  10. 42有连线。2没被访问过，访问2。
  11. 24有连线。4被访问过。
  12. 26有连线。6没被访问过。访问6。
  13. 60有连线，0被访问过。
  14. 62有连线，2被访问过。
  15. 63有连线，3没被访问过，访问3。
  16. 36有连线，6被访问过。
  17. 退回6。（60、62、63均被处理过）
  18. 退回2。（24、26也均被处理过）
  19. 退回4。（40、41、42均处理过）
  20. 退回0。（04、06均处理过）
  21. 遍历结束。

我们发现了规律，对于每个节点，要做的事情都是有规律的。我们把这个要做的事情，写在一个函数dfs里。函数dfs功能大概就是找自己的邻接点，看看他们是不是访问过，有能访问的就继续让自己的邻接点调用dfs，没有能访问的就返回。

void dfs(int cur)
{
    for (int node = 0; node < node_num; node++) //对于每个节点
    {
        //判断当前顶点cur和顶点i是否是邻接点,并判断顶点i是否已经访问过
        if (line[cur][node] == 1&& visited[node] == 0)
        {
            visited[node] = 1; //该点的访问标志位置1
            dfs(node);  // 递归调用，继续往下找路
        }
    }
    return;   //所有点都访问过或者递归回来了，直接返回。
}

递归的缺点也是很明显的。如果图很复杂，就会不断重复调用dfs，导致函数一直返回不了。因为每调用一个方法就会在栈上创建一个栈帧，方法调用结束后就会弹出该栈帧，而栈的大小不是无限的，所以递归调用次数过多的话就会导致栈溢出。

深度优先遍历可以直接通过栈来实现，具体的可以看下面的链接的动图。

图文详解两种算法：深度优先遍历（DFS）和广度优先遍历（BFS）

匈牙利算法匹配流程

看完上面，我们大概对图的深度遍历（DFS）有一个认识了。可以看到深度遍历找到了很多路径，对图上的节点全部搜索了一遍，但是并不期望这些路径有什么特别之处，比如最短路径。深度遍历是很多图算法的基础。

而深度遍历DFS也是匈牙利算法的基础。

  1. 当前配对目标x0。 x0y0有连线，且y0没有被配对。因此x0和y0配对。
  2. 当前配对目标x1。x1y0有连线，y0已被配对。

此时，有两种方案：（1）通过增广路解决冲突。得到增广路径x1–y0–x0–y2。（2）选择新的配对对象。x1和y1也有连线，x1和y1配对。

从图的遍历的角度来看，方案1优先使用增广路，意味着优先遍历下一层的节点，这种方案采用了深度优先的思路。

而方案2优先遍历同一层的另一节点，是一种广度优先的思路。

按照深度优先的思路，代码如下：

#include <iostream>
#include <cstdio>
#include <cstring>
#include "main.h"

using namespace std;

#define DEBUG_FLAG 1

int     nx, ny;
int     line[MAX_NODE][MAX_NODE];
int     visited[MAX_NODE];
int     cx[MAX_NODE], cy[MAX_NODE];

// with printf, printf some information for understanding this code.
int find_path(int u){
    for(int v = 0; v < ny; v++){
#if DEBUG_FLAG
        printf("y%d is checked\n", v);
        if(line[u][v]){
            printf("there is a line between x%d and y%d\n", u, v);
            if(!visited[v]){
                printf("y%d is not visited! \n", v);
            }
            else{
                printf("y%d has been visited! \n", v);
            }
        }
        else{
            printf("there is no line between x%d and y%d\n", u, v);
        }
#endif
        if(line[u][v] && !visited[v]){
            visited[v] = 1;
#if DEBUG_FLAG
            if(cy[v]==-1){
                printf("y%d is not matched\n", v);
            }
            else{
                printf("y%d has been matched\n", v);
                printf("call find function, find another match for x%d\n", cy[v]);
            }
#endif
            if(cy[v]==-1 || find_path(cy[v])){
#if DEBUG_FLAG
                printf("x%d and y%d is matched!\n", u, v);
                printf("function find return 1\n");
#endif
                cx[u] = v;
                cy[v] = u;
                return 1;
            }
        }
    }
#if DEBUG_FLAG
    printf("function find return 0\n");
#endif
    return 0;
}

int hungarian(){
    int matchNum=0;
    for(int i=0; i<nx; i++){
#if DEBUG_FLAG
        printf("find match for x%d\n", i);
#endif

        if(cx[i]==-1){
#if DEBUG_FLAG
            printf("x%d is not matched\n", i);
#endif
            memset(visited, 0, sizeof(visited));
            matchNum+=find_path(i);
        }
    }
    return matchNum;
}
int main()
{
    nx = 4;
    ny = 3;
    line[0][0] = true;   line[0][1] = false;  line[0][2] = true;
    line[1][0] = true;   line[1][1] = true;   line[1][2] = false;
    line[2][0] = true;   line[2][1] = false;  line[2][2] = true;
    line[3][0] = false;  line[3][1] = false;  line[3][2] = true;
    memset(cx, -1, sizeof(cx));
    memset(cy, -1, sizeof(cy));

    // call hungarian algorithm
    int matchNum = hungarian();

    // printf result on screen
    printf("%d pairs of matched nodes found!\n", matchNum);
    printf("===========Hungarian Algorithm Match Result===============\n");
    for(int i=0; i<nx; i++){
        if(cx[i]==-1){
            printf("x[%d]--->None\n", i);
        }
        else{
            printf("x[%d]--->y[%d]\n", i, cx[i]);
        }
    }
    return 0;
}

main函数中， 一开始nx是左侧集合的节点个数，ny是右侧节点个数。line[][]是二分图，true表示有连线，false表示没有连线。cx[]储存的是左侧节点配对结果，-1表示还没匹配。 cy[]是右侧节点的配对情况，-1表示没有配对。
