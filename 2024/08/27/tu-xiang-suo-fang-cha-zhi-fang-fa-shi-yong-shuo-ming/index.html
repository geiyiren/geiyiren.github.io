图像缩放插值方法使用说明

图像缩放 插值方法
图像处理

更新日期:   2024-09-19

方法概览

  • 图像加载
库 函数 默认加载图像格式
OpenCV imread BGR
PIL open RGB
torchvision.datasets ImageFolder RGB

ImageFolder类通过Pillow（PIL）库来加载图像，而Pillow加载图像的默认模式是RGB。因此，使用ImageFolder加载图像时，它们会以RGB格式加载。

  • 尺寸变换
库 函数 默认加载图像格式
OpenCV resize 双线性插值INTER_LINEAR
PIL Resize 根据输入图像格式确定
torchvision.transforms Resize 双线性插值InterpolationMode.BILINEAR

注意：opencv 的INTER_NEAREST_EXACT比特精确近邻取样插值方法的结果和PIL的NEAREST最近邻算法的结果是一致的。

OpenCV图像缩放

  • opencv默认的图像缩放插值方法为双线性插值INTER_LINEAR。
  • 支持的插值方法如下：
插值方法 说明
INTER_NEAREST 近邻取样插值
INTER_LINEAR 双线性插值
INTER_CUBIC 双三次插值
INTER_AREA 区域插值。使用像素区域关系的重采样。它可能是一种首选的图像抽取方法，因为它提供了莫尔’免费的结果。但是当缩放图像时，它类似于 INTER _ NEAREST 方法。
INTER_LANCZOS4 Lanczos插值。8乘以8的 Lanczos 插值
INTER_LINEAR_EXACT Bit exact bilinear interpolation
INTER_NEAREST_EXACT 比特精确近邻取样插值。这将产生与 PIL、 scikit-image 或 Matlab 中最近邻法相同的结果
WARP_FILL_OUTLIERS flag:填充所有目标图像像素。如果其中一些对应于源图像中的异常值，则将其设置为零
WARP_INVERSE_MAP flag, inverse transformation

torchvision.transforms图像缩放

  • torchvision.transforms.Resize函数可以实现图像的缩放，默认插值方法为双线性插值InterpolationMode.BILINEAR。

  • 如果输入为张量，则仅支持临近插值InterpolationMode.NEAREST, 双线性插值InterpolationMode.BILINEAR 和双三次插值InterpolationMode.BICUBIC

  • 也支持PIL库的插值方法,例如： transforms.Resize(args.resolution, interpolation=PIL.Image.BILINEAR)

PIL的图像缩放

  • 默认的插值方法会根据输入图像的不同而有所不同：
输入图像格式 差值方法
1 Resampling.NEAREST
P Resampling.NEAREST
such as: I;16 and so on Resampling.NEAREST
Otherwise Resampling.BICUBIC

图像格式说明

  • 1 模式

表示二值图像，即每个像素要么是黑（值为0），要么是白（值为255）。
每个像素用1位表示。

  • P 模式：

表示调色板图像，使用一个调色板将每个像素映射到实际的颜色。
每个像素用8位表示，颜色数据由一个调色板来定义，调色板中最多可以有256种颜色。
这两种模式在处理不同类型的图像时各有用途。例如，”1” 模式常用于需要非常简单的黑白图像，而 “P” 模式则适用于图像颜色相对有限的情况下。

  • the image mode specifies a number of bits, such as “I;16”
    the default filter isResampling.NEAREST.

  • Otherwise
    the default filter isResampling.BICUBIC.

为双线性插值Resampling.BICUBIC

  • 其他方法如下：
    • PIL.Image.NEAREST (或 InterpolationMode.NEAREST)：
      最近邻插值。速度快，但图像质量较低。
    • PIL.Image.BILINEAR (或 InterpolationMode.BILINEAR)：
      双线性插值。效果较好，图像更平滑。
    • PIL.Image.BICUBIC (或 InterpolationMode.BICUBIC)：
      双三次插值。生成的图像质量更好，但计算量较大。
    • PIL.Image.LANCZOS (或 InterpolationMode.LANCZOS)：
      Lanczos插值。适用于高质量的图像处理，计算量大，但生成的图像质量最高。
    • PIL.Image.BOX (或 InterpolationMode.BOX)：
      简单的平均插值，通常用于图像缩小。
    • PIL.Image.HAMMING (或 InterpolationMode.HAMMING)：
      Hamming插值，一种平滑插值方法。
      def resize(self, size, resample=None, box=None, reducing_gap=None):
       """
       Returns a resized copy of this image.

       :param size: The requested size in pixels, as a 2-tuple:
          (width, height).
       :param resample: An optional resampling filter.  This can be
          one of :py:data:`Resampling.NEAREST`, :py:data:`Resampling.BOX`,
          :py:data:`Resampling.BILINEAR`, :py:data:`Resampling.HAMMING`,
          :py:data:`Resampling.BICUBIC` or :py:data:`Resampling.LANCZOS`.
          If the image has mode "1" or "P", it is always set to
          :py:data:`Resampling.NEAREST`. If the image mode specifies a number
          of bits, such as "I;16", then the default filter is
          :py:data:`Resampling.NEAREST`. Otherwise, the default filter is
          :py:data:`Resampling.BICUBIC`. See: :ref:`concept-filters`.
       :param box: An optional 4-tuple of floats providing
          the source image region to be scaled.
          The values must be within (0, 0, width, height) rectangle.
          If omitted or None, the entire source is used.
       :param reducing_gap: Apply optimization by resizing the image
          in two steps. First, reducing the image by integer times
          using :py:meth:`~PIL.Image.Image.reduce`.
          Second, resizing using regular resampling. The last step
          changes size no less than by ``reducing_gap`` times.
          ``reducing_gap`` may be None (no first step is performed)
          or should be greater than 1.0. The bigger ``reducing_gap``,
          the closer the result to the fair resampling.
          The smaller ``reducing_gap``, the faster resizing.
          With ``reducing_gap`` greater or equal to 3.0, the result is
          indistinguishable from fair resampling in most cases.
          The default value is None (no optimization).
       :returns: An :py:class:`~PIL.Image.Image` object.
       """

特别注意

当opencv和PIL同时使用双线性插值时得到的图像也存在差异

import cv2
from PIL import Image
import numpy as np
image_path = '20240826_191429236236952.png'
image_size = (64,64)
image_PIL = Image.open(image_path).convert('RGB')
image_PIL = image_PIL.resize(image_size, Image.BILINEAR)
image_PIL = np.array(image_PIL, dtype=np.float32)

image_cv = cv2.imread(image_path)
image_cv = cv2.resize(image_cv, image_size, interpolation=cv2.INTER_LINEAR)
image_cv =  cv2.cvtColor(image_cv,  cv2.COLOR_BGR2RGB)
image_cv = image_cv.astype(np.float32)

# 计算两张图像的差值
difference = cv2.absdiff(image_PIL, image_cv)
# 显示差值图像
cv2.imshow('Difference', difference)
cv2.waitKey(0)
cv2.destroyAllWindows()
print(difference)
