yolo训练参数解释

yolo 训练

更新日期:   2025-08-29

Train

参数名 类型 默认值 说明
model str (optional) 模型文件路径，如 yolov8n.pt 或配置文件 yolov8n.yaml；需要进行resume时，这里需要填写需要加载模型的权重
data str (optional) 数据集配置文件路径，如 coco8.yaml
epochs int 100 训练轮数
time float (optional) 指定训练时长（小时），若设定则覆盖 epochs
patience int 100 若连续该轮数没有提升，则提前停止训练
batch int 16 每批训练图像数量， -1 表示自动调整
imgsz int list 640
save bool True 是否保存训练检查点和预测结果
save_period int -1 每多少 epoch 保存一次模型， -1 表示禁用
cache bool str False将数据集加载后保存在内存中或disk上，可以适当提升训练速度
device int str list
workers int 8 加载数据时的线程数（每个 DDP rank）
project str (optional) 项目名称
name str (optional) 实验名称，结果将保存至 project/name 目录下
exist_ok bool False 若目录已存在，是否覆盖
pretrained bool str True
optimizer str auto 优化器选择，选项有：SGD, Adam, Adamax, AdamW, 等
verbose bool True 是否打印详细信息
seed int 0 随机种子，确保可复现性
deterministic bool True 是否启用确定性训练，主要用于确保每次训练结果一致，在调试、复现实验结果时非常重要
single_cls bool False 是否将多类别数据作为单类别训练，model.nc 会被设置为 1，配置文件中配置的多类信息失效
rect bool False 是否使用矩形训练/验证
cos_lr bool False 是否使用余弦学习率调度器
close_mosaic int 10 最后多少个 epoch 禁用 mosaic 数据增强
resume bool False 恢复训练是否从上次检查点恢复训练
amp bool True 是否启用自动混合精度（AMP）训练
fraction float 1.0 训练集使用比例（如0.1表示只使用10%的训练集）
profile bool False 是否在训练期间测试 profile ONNX 或 TensorRT 推理速度
freeze int list 训练的时候冻结某些层
multi_scale bool False 是否启用多尺度训练

Val/Test

参数名 默认值 类型 说明
val True bool 是否在训练时进行验证/测试
split val str 用于验证的数据集划分（val/test/train）
save_json False bool 是否将结果保存为 JSON 文件
conf float 目标检测置信度阈值（默认：预测时0.25，验证时0.001）
iou 0.7 float NMS 的 IoU 阈值在进行nms时iou大于该值的框会被认为时重复框，进而被删除
max_det 300 int 每张图像的最大检测数
half False bool 是否使用半精度 FP16
dnn False bool 是否使用 OpenCV DNN 模式推理 ONNX
plots True bool 是否在训练/验证过程中保存图像和图表

Predict

参数名 默认值 类型 说明
source str 输入图像或视频路径
vid_stride 1 int 视频帧步长
stream_buffer False bool 是否缓冲所有视频流帧
visualize False bool 是否可视化模型特征
augment False bool 是否对预测输入进行增强
agnostic_nms False bool 类别无关的 NMS
classes int / list[int] 仅保留指定类别的检测结果
retina_masks False bool 使用高分辨率分割掩码
embed list[int] 输出指定层的特征向量

Visualize

参数名 默认值 类型 说明
show False bool 是否展示预测图像/视频（前提环境支持）
save_frames False bool 是否保存预测视频的帧
save_txt False bool 是否保存结果为 txt 文件
save_conf False bool 是否保存置信度得分
save_crop False bool 是否保存裁剪出的目标图像
show_labels True bool 是否显示预测类别标签
show_conf True bool 是否显示预测置信度
show_boxes True bool 是否显示预测框
line_width int 边框线宽（默认根据图像大小自动缩放）

Export

参数名 默认值 类型 说明
format torchscript str 导出格式
keras False bool 是否导出为 Keras 格式
optimize False bool TorchScript 导出是否优化
int8 False bool CoreML / TF 是否使用 INT8 量化
dynamic False bool ONNX/TF/TensorRT 是否使用动态轴
simplify True bool ONNX 模型简化
opset int ONNX 的 opset 版本
workspace float TensorRT 的工作空间（GiB）
nms False bool CoreML 中是否添加 NMS

超参数

参数名 默认值 类型 说明
lr0 0.01 float 初始学习率（SGD=1E-2，Adam=1E-3）
lrf 0.01 float 最终学习率，lr0 × lrf
momentum 0.937 float SGD 动量 / Adam β1
weight_decay 0.0005 float 权重衰减
warmup_epochs 3.0 float 预热轮数
warmup_momentum 0.8 float 预热初始动量
warmup_bias_lr 0.1 float 预热初始 bias 学习率
box 7.5 float 目标框损失权重
cls 0.5 float 类别损失权重
dfl 1.5 float DFL 损失权重
pose 12.0 float 姿态估计损失权重
kobj 1.0 float 关键点目标损失权重
nbs 64 int 名义 batch size
  • 图像增强参数
    参数名 默认值 类型 说明
    hsv_h 0.015 float HSV 色调增强
    hsv_s 0.7 float HSV 饱和度增强
    hsv_v 0.4 float HSV 明度增强
    degrees 0.0 float 旋转角度
    translate 0.1 float 平移
    scale 0.5 float 缩放
    shear 0.0 float 裁剪倾斜角度
    perspective 0.0 float 透视变换
    flipud 0.0 float 上下翻转概率
    fliplr 0.5 float 左右翻转概率
    bgr 0.0 float BGR 通道扰动概率
    mosaic 1.0 float mosaic 数据增强概率
    mixup 0.0 float mixup 概率
    cutmix 0.0 float cutmix 概率
    copy_paste 0.0 float segment 拷贝粘贴增强概率
    copy_paste_mode flip str 拷贝粘贴方式（flip, mixup）
    auto_augment randaugment str 图像自动增强策略
    erasing 0.4 float 分类训练时随机擦除概率（< 1.0）。在训练过程中，对图像的随机区域进行遮挡（擦除），增强模型鲁棒性
